# HDC2022-DevelopmentBoard
 ---------------------------------------
 TI HDC2022 Development Board & Firmware


## Hardware

* Temperature and Humidity Sensor - HDC2022DEPR
* Internal LDO TLV74018PDBVR (1.8V)
* Expansion header for I2C and Power Pins
* PCB size 25x15 mm
